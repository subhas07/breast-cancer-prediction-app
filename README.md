# 🩺 Breast Cancer Prediction System

An end-to-end Machine Learning web application that predicts whether a breast tumor is **Benign** or **Malignant** using medical diagnostic features.

Built using:

* Python
* Scikit-learn
* Streamlit
* Plotly

# 📌 Project Overview

This project uses Machine Learning to analyze breast cancer diagnostic data and predict tumor type.

The application provides:

* Real-time prediction
* Prediction probability analysis
* Interactive charts
* Modern responsive UI
* User-friendly medical input interface

# 🤖 ML Algorithm Used

## Support Vector Machine (SVM)

The Support Vector Machine classifier was used for prediction because of its strong performance on classification tasks and high accuracy for medical datasets.

---

# 📈 Features Used

The model uses the following medical features:

* radius_mean
* texture_mean
* perimeter_mean
* area_mean
* smoothness_mean
* compactness_mean
* concavity_mean
* concave points_mean
* symmetry_mean
* fractal_dimension_mean
